Changing The Main Color

Shrimpy HTML template enables you to set your desired color as the main color of your website. In default the main color code is #01aaae and you can easily change it.

To change the main color (accent color), open style.css file. This file is located in the root, where the index.html is located. Now go to line 101 - 133. You can see the elements are grouped in order to change easily.

Slideshow

Shrimpy uses ResponsiveSlides jquery slideshow plugin. You can find it here (http://responsiveslides.com). It's very easy to work with it. You just need to find the div with id of #slideshow, and create li elements inside it. You need to follow the sample slides markup.

Navigation Menu

To change the main menu items, find the #navigation and #mobile-navigation HTML id names and then set your desired texts and href values there. Note that both of "navigation" and "mobile-navigation" should be the same. Because "navigation" appears in bigger screens like desktop computers and the "mobile-navigation" appears in small screen devices like mobiles.
Since Shrimpy is a one page website, you will need to link each menu item to a section by # characters.

About Section

The id of the section is #about. It has 2 columns. The left column is a slider containing a few slides.
The right column contains a tab and accordion section. It's very straitforward to change the text values.

Daily Specialties

This section can be used as image gallery. The ID is #special-gallery. Each gallery item has a wrapper with class of special-gallery-item. Also the overlay text is inside a div with the class name of special-gallery-overlay.

SeaFood Dishes

This is another section can be used as gallery. The ID is #gallery
Each item has a wrapper with bootstrap 3 col classes. .

Our Menu

This section contains a big tab. The ID for this menu is #big-menu.
The tab titles are inside .special-tabs-titles ul and each tab content is inside .special-tab-content div.
Each tab content has a 3 columns gallery. The columns are based on Bootstrap, and inside each column you will find .food-menu-item div.

Our Cooks

This section ID is #cooks. It's a simple 4 columns bootstrap. You can put the coock image inside .cook-face div.

Blog

The section ID is #blog. It's a 2 column layout by bootstrap grid. The left column is the image container, and the right one is for blog post title, meta, and content.

Contacu Us

To setup the contact form, you need to open action.php. Go to line 6 and change your_email@gmail.com with your own email address. Also you can change the error messages too.
